============================
Download MediaElch for macOS
============================

macOS 10.13 or later is supported.  You can download MediaElch as a ``*.dmg`` file at:

+-------------------+--------------------+-----------------------------------------------+
| System (64 bit)   | Version            | Download                                      |
+===================+====================+===============================================+
| macOS 11 or later | 2.10.0  (GitHub)   | |gh_pages_release|                            |
+-------------------+--------------------+-----------------------------------------------+
|                   | 2.10.0  (Mirror)   | |mirror_release|                              |
+-------------------+--------------------+-----------------------------------------------+
|                   | latest (unstable)  | |mirror_nightly_macOS_11_or_later|            |
+-------------------+--------------------+-----------------------------------------------+
| macOS X           | 2.10.0  (GitHub)   | |gh_pages_release|                            |
+-------------------+--------------------+-----------------------------------------------+
|                   | 2.10.0  (Mirror)   | |mirror_release|                              |
+-------------------+--------------------+-----------------------------------------------+
|                   | latest (unstable)  | |mirror_nightly_macOS_X|                      |
+-------------------+--------------------+-----------------------------------------------+


.. warning::

   The DMG for macOS 11 or later is currently about 180MiB.
   With the next release, it will be down to 70MiB again.


*See also section macOS in:* :doc:`Install MediaElch <../installation>`.
