=========================================================
Build MediaElch
=========================================================

MediaElch can be build using ``qmake`` and ``CMake``.
Using `QtCreator`_ is supported and advised.

CMake is supported as well. Following instructions will use
QMake, though.

*Supported platforms:*

   :maxdepth: 1

   macOS <macos>
   Ubuntu 18.04 LTS and 20.04<ubuntu_18_04>
   Ubuntu 21.04 LTS and later<ubuntu_21_04>
   openSUSE Leap 42.3 and later<openSUSE_Leap_42_3>
