MediaElch can be build using qmake and CMake. Using QtCreator is supported and advised.

We support Qt 5 and Qt 6.

CMake is supported as well. Following instructions will use QMake, though.

Supported platforms:

  • Windows
  • macOS
  • Ubuntu 18.04 LTS and 20.04
  • Ubuntu 22.04 LTS and later
  • openSUSE Leap 15 and later
  • Fedora 36 and later
